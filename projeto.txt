PLANEJAMENTO DO SITE

    *PORTIFÓLIO

 2- QUAL INFORMAÇÃO VOCÊ QUER APRENSENTAR DENTRO DO ASSUNTO
    *ESCREVA PARAGRAFOS PENSE EM UMA IMAGEM PARA ILUSTRAR SUA PAGINA

 3- COMO SEU SITE DEVE PARECER?
    *CORES DE FUNDO

 4- RASCUNHE COMO O PROJETO DO SEU SITE DEVE SE PARECER 
    *EM FOLHA DE PAPEL
 
 5- MANTENHA SEUS TEXTOS POR PERTO 
    *TITILOS
    *PARAGRAFOS

 6- SELECIONE A COR DO TEMA
    *#ABB5B9 (HEXADECIMAL)

 7- SELECIONE A IMAGEM QUE SE ENCAIXE NO SEU SITE
    *SALVE COMO E DEIXE GUARDADO EM ALGUM LUGAR SEGURO
    *COPIE TAMBÉM O ENDEREÇO WEB COMO ALTERNATIVA PARA USO POSTERIOR
    *PARA SE PROTEGER DE DIREITOS AUTORAIS DE IMAGEM FILTRE EM:
        GOOGLE IMAGENS -> FERRAMENTAS -> MARCADA PARA REUTILIZAÇÃO

 8- SELECIONE A FONTE NO GOOGLE FONTES
    *<link href="https://fonts.googleapis.com/css2?family=Comic+Neue:ital,wght@1,700&display=swap" rel="stylesheet">
    *CSS: font-family: 'Comic Neue', cursive;

 9- NA PASTA DO PROJETO DEVE CONTER:
    * PASTA PARA IMAGENS
    * PASTA PARA SCRIPTS
    * PASTA PARA CSS
